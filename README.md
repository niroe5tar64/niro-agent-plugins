# niro-agent-plugins

Claude Code向けのプラグイン集。

## インストール方法

```
/plugin marketplace add niroe5tar64/niro-agent-plugins
```

## 含まれるプラグイン

### [git-ops](./src/git-ops/)

Git操作を効率化するプラグイン。

- `/commit` - ステージされた変更から適切なコミットメッセージを生成

### [statusline](./src/statusline/)

カスタムステータスラインを提供するプラグイン。

- モデル・ディレクトリ・Git情報・コンテキスト使用量・コストを表示
