# niro-agent-plugins

Claude Code向けのプラグイン集。

## インストール方法

```
/plugin marketplace add niroe5tar64/niro-agent-plugins
```

## 含まれるプラグイン

### [git-ops](./git-ops/)

Git操作を効率化するプラグイン。

- `/commit` - ステージされた変更から適切なコミットメッセージを生成

### [decision-support](./decision-support/)

意思決定を支援するプラグイン。

- `/discussion` - 1対1の壁打ちで思考を整理
- `/discussion-forum` - 複数ロールによる多角的な議論
